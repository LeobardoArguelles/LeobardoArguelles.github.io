Hydra (bruteforcing)

Tool to bruteforce logins/passwords. Basic syntax: hydra -l root -P <wordlist> <service>://<target>:<port> <flags> # E.g: hydra -l root -P /usr/share/wordlists/metasploit/unix_passwords.txt ssh://192.168.57.134:22 -t 4 -V -l is for loggin -P indicates the password file -t is the number of threads -V verbosity

Idempotente (Ley de álgebra booleana)

Una variable, operada consigo misma, genera esa misma variable: \(A \vee A = A\) \(A \wedge A = A\) TABLAS DE VERDAD Para comprobar. OR \(Y = A \vee A\) A A Y 0 0 0 1 1 1 AND \(Y = A \wedge A\) A A Y 0 0 0 1 1 1

Identidad (Ley de álgebra booleana)

Se mantiene el valor de la variable, pues realiza una operación con el valor neutral de esa operación: \(A+0=A\) Las siguientes formas son equivalentes: \(A\; or\; 0 = A\) \(A \vee 0 = A\) \(A*1=A\) Las siguientes formas son equivalentes: \(A\; and\; 1 = A\) \(A \wedge 1 = A\)

Ingeniere

Es un profesional que inventa, diseña, analiza, construye, y prueba máquinas, sistemas complejos, estructuras, y materiales para cumplir objetivos funcionales, al mismo tiempo que considera limitantes impuestas por regulaciones, practicicidad, seguridad, y costo. Ingenieria en Computacion

Se enfoca en usar principios de ingeniería para resolver problemas usando hardware de computadora. Temas que cubre: Arquitectura de computadoras, redes, seguridad. Planificación, diseño, construcción, probar dispositivos que realicen una función específica. Combina física, ingeniería eléctria y electrónica, e informática. Más hardware que software. COMPUTACIÓN Proviene del latín computatĭo, y se deriva de computare, que quiere decir computar o evaluar. Ciencia que se ocupa del estudio de las computadoras, abarcando su diseño, operación, y uso en el procesamiento de datos.

Insertion sort

Cada elemento se examina y se coloca en la posición correcta del subarray “ya ordenado”. Nota: Las condiciones iniciales consideran al elemento con índice 0 como el “subarray ordenado”. Análisis Peor caso: \(O (n^2)\) Mejor caso: \(O (n)\) Caso promedio: \(O (n^2)\) Pseudocódigo for i=1 to i < array_size: aux = numbers[i] a = i - 1 # Insertar numbers[a] en el subarray ordenado while a >= 0 and numbers[a] > aux: numbers[a+1] = numbers[a] a -= 1 numbers[a+1] = aux

Install Matlab Arch Linux

Download, install, and activate Matlab from the official page. There shouldn’t be any problems here. Now, what I had to do to actually make it work: Install Java 8 sudo pacman -S jdk8-openjdk. Add to your .bashrc these variables: export MATLAB_JAVA=/usr/lib/jvm/java-8-openjdk/jre/ export J2D_D3D=false export GTK_PATH=/usr/lib/gtk-3.0 export _JAVA_AWT_WM_NONREPARENTING=1 Create a symbolic link to launch it just calling matlab. ln -s <path/to/MATLAB/VERSION/bin/matlab> /usr/local/bin E.g: ln -s /usr/local/MATLAB/R2020a/bin/matlab /usr/local/bin Done?

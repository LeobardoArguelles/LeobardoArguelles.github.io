Use the program `xset`.


## COMMON COMMANDS {#common-commands}

-   To set time before screensaver activation, use: `xset s <seconds>`
-   To turn off the screensaver, use: `xset s off`.
-   Prevent screen from blacking: `xset s off -dpms`

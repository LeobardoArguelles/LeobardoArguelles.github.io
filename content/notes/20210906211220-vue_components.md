Vue components have 3 parts:

-   <template>
-   <script>
-   <style>


## TEMPLATE {#template}

It has the markup and display logic.


## SCRIPT {#script}

It has the non-display logic.
It must have a default exported JS object.


## STYLE {#style}

It has the css.
